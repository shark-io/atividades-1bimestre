Esta página é dedicada a aula sobre o atributo div, durante a aula de Programação de Computadores 2, no dia 24/03, com o auxílio da professora Elisandra. A aula teve seu foco destinado a ensinar os alunos do 2.°Informática(T2) a utilizar o atributo div, que consiste em ser utilizado como uma caixa para agrupar conteúdos.

Ao utilizar, apenas usando a estrutura html, você não percebe que há diferença de ter ou não uma div, porém irá fazer mais
sentido utilizando a linguagem de programação CSS, onde serão atribuidos estilos, deixando mais agradável visualmente.
